# クォータニオン実感プログラム

クォータニオンとオイラー角の相互変換をリアルタイムで可視化し、クォータニオンのメリット（ジンバルロックの回避など）を体感できるWebアプリケーション。

## 機能

- **3Dビュー**: 矢印形オブジェクトの回転をリアルタイム表示（座標軸・グリッド付き）
- **クォータニオン入力**: w, x, y, z パラメータをスライダー/数値で操作
- **オイラー角入力**: Roll, Pitch, Yaw をスライダー/数値で操作
- **双方向変換**: どちらかを変更すると自動的にもう一方に変換
- **ジンバルロック警告**: Pitch が ±90° 付近で警告表示

## セットアップ

```bash
# 依存関係インストール
uv sync

# アプリ起動
uv run python main.py
```

ブラウザで http://127.0.0.1:5000 を開く

## 技術スタック

- **バックエンド**: Python + Flask
- **フロントエンド**: Three.js (3D表示)
- **数学**: scipy.spatial.transform.Rotation
